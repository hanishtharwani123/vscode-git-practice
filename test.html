Git Branching Guide

What is a Branch?

A branch in Git is simply a lightweight movable pointer to one of these commits. The default branch name in Git is master. As you start making commits, you're given a master branch that points to the last commit you made. Every time you commit, it moves forward automatically.

Creating a New Branch

To create a new branch, you can use the following command:

git branch new-branch-name

Switching Branches

To switch to an existing branch, you can use the following command:

git checkout branch-name

Creating and Switching to a New Branch

You can create and switch to a new branch in one command:

git checkout -b new-branch-name

Merging Branches

To merge a branch into your current branch, you can use the following command:

git merge branch-name

Deleting a Branch

To delete a branch, you can use the following command:

git branch -d branch-name
